# Rixco

A Brazilian-inspired icon set for designers and developers.

**Rixco** combines minimalist design with technical precision. 250+ line-style icons, stroke-based and fully customizable — ready for any project.

## Features

- ✨ **250+ Icons** — Growing collection of carefully crafted line-style icons
- 🎨 **Minimalist Design** — Clean, geometric, stroke-based approach
- 🔧 **Fully Customizable** — Adjust stroke width, color, and scale with CSS
- 📦 **Multiple Formats** — Webfont, SVG, NPM package (coming soon)
- 🌍 **Brazilian Identity** — Inspired by Brazilian design heritage
- 📱 **Responsive** — Perfect at any size, from 16px to 256px+
- ♿ **Accessible** — Built with semantic HTML and ARIA support
- 📄 **Open Source** — MIT licensed, free for commercial use

## About the Name

**Rixco** comes from the Portuguese word *"risco"* (stroke, line, trace). It represents precision, artistry, and the craftsmanship behind each icon.
